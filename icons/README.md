# Extension Icons

Place your extension icons in this folder:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Quick Icon Creation

You can create simple placeholder icons using any image editor or online tool. For production, use the Cerby logo or your brand icon.

## Icon Requirements

- Format: PNG
- Background: Transparent or solid color
- Size: Exact dimensions as specified
- Aspect ratio: 1:1 (square)
