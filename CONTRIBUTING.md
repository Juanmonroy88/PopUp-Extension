# Contributing

## Version control

- Use meaningful commit messages (e.g. "Fix TOTP field border" rather than "fix stuff").
- Commit when things work; test in Chrome before committing.
- Update [CHANGELOG.md](CHANGELOG.md) for user-facing changes (Added / Fixed / Changed).

For more detail, see [VERSION_CONTROL.md](VERSION_CONTROL.md).

## Setup

1. Clone the repo and open the project folder in Chrome as an unpacked extension (`chrome://extensions` → Load unpacked).
2. No build step: edit HTML/CSS/JS and reload the extension to test.

## Scope

This repo is **Chrome only**. Safari and other browsers are out of scope.
